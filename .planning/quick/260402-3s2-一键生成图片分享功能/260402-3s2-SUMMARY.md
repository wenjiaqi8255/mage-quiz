   - Maintains existing design system

### Technical Approach

**Clone-based Capture Pattern** (Critical for html2canvas):
```tsx
const clone = element.cloneNode(true) as HTMLElement
clone.style.position = 'absolute'
clone.style.opacity = '1'
clone.style.visibility = 'visible'
clone.style.left = '0'
clone.style.top = '0'
document.body.appendChild(clone)

const canvas = await html2canvas(clone, { scale: 2, backgroundColor: '#0a0a0b' })
document.body.removeChild(clone)
```

This pattern avoids common html2canvas issues:
- Blank images from hidden elements
- Zero scrollHeight from `visibility: hidden`
- Positioning artifacts from `position: fixed`
